# DFT Logística — Estratégia de Google Ads para atrair contrato recorrente
### Pronto para colar no Google Ads · Cardô Marketing · Junho/2026

> **Diagnóstico:** as keywords genéricas ("transportadora SP", "transportadora São Paulo") têm **intenção de carga avulsa/spot**. Por isso chegam leads, mas só carga picada — nenhum contrato recorrente. A correção tem 3 frentes: **negativar o avulso**, **trocar as keywords pela intenção do comprador recorrente** e **usar o anúncio para desqualificar** quem é pontual.
>
> ⚠️ Conferir limites de caractere no editor do Google Ads (título ≤ 30, descrição ≤ 90, callout ≤ 25, caminho ≤ 15).

## 2. 🎯 Nova estrutura de palavras-chave (por intenção do ICP)

Sair do termo genérico de cidade e ir para a **intenção de quem contrata operação recorrente**. Usar **frase** e **exata** (evitar ampla, que foi a fonte do desperdício).

### Grupo A — Operador logístico / Terceirização
```
"operador logístico"
"operador logístico em sp"
"logística terceirizada"
"terceirização logística"
"operador logístico para indústria"
"empresa de logística e distribuição"
[operador logístico b2b]
```

### Grupo B — Transporte dedicado / Frota dedicada
```
"transporte dedicado"
"transportadora dedicada"
"frota dedicada"
"transporte rodoviário dedicado"
"transporte fracionado para empresas"
[transporte dedicado sp]
```

### Grupo C — Armazenagem e distribuição
```
"armazenagem e distribuição"
"operador logístico armazenagem"
"centro de distribuição terceirizado"
"armazém credenciado"
"armazenagem para indústria"
```

### Grupo D — Por segmento (cargas de alto valor)
```
"transportadora para eletrônicos"
"transporte de cargas de alto valor"
"transportadora para cosméticos"
"transportadora para farmacêutico"
"transporte de medicamentos"
"logística para indústria farmacêutica"
"transportadora para alimentos"
```

### Grupo E — Branding (manter)
```
[dft logística]
"dft transportadora"
"dft logistica sp"
```

> **Lances:** começar com **Maximizar conversões** depois de acumular ~15-20 conversões qualificadas; até lá, **CPC manual** com teto de R$6-8 nos grupos A-D (intenção mais cara, porém certa).

## 5. ✅ Plano de implementação

1. Pausar/encerrar o grupo `[G01] Demanda` genérico (ou ao menos as keywords amplas tipo "transportadora SP", "carretos sp").
2. Subir as **negativas** no nível da campanha.
3. Criar os **grupos A-E** com as novas keywords (frase/exata).
4. Subir os **3 RSAs** (um por tema) + extensões.
5. Apontar todos para a **nova LP**: https://cardo-jpg.github.io/dft-lp/combinada/
6. Revisar **Termos de Pesquisa** toda semana e negativar o que vier de avulso.
7. Em paralelo, considerar **LinkedIn Ads** (diretores de supply chain) — é onde o comprador recorrente realmente está; o Search, mesmo otimizado, traz mais carga pontual por natureza.

> **Expectativa realista:** com verba enxuta (~R$3.000/mês) e CPC do setor ~R$5, o volume de leads vai **cair** — mas a **qualificação sobe**. O objetivo deixou de ser "muitos leads" e passou a ser "leads de contrato recorrente".
